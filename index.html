Number Baseball Game

It is a game of matching 3 numbers and positions.
1) Strike: If the number and position match
2) Ball: If the number is correct but the position is wrong
3) 3Out: If all 3 numbers are missing
You can choose 0 to 9 and repeated number is not permitted.
You have nine chances. Good luck.
